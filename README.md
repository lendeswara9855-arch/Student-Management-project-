# 🎓 Student Result Management System

A simple **Python-based Student Result Management System** that allows users to manage student records efficiently through a command-line interface (CLI). This project demonstrates the use of **Python functions, lists, dictionaries, loops, and conditional statements** for performing CRUD operations.

---

## 📌 Features

✅ Add New Student Records
✅ Display All Students
✅ Search Student by Roll Number
✅ Update Student Marks & Grades
✅ Delete Student Records
✅ Automatic Percentage Calculation
✅ Automatic Grade Generation
✅ Menu-Driven CLI Program

---

## 🛠️ Technologies Used

* Python 3.x
* Command Line Interface (CLI)
